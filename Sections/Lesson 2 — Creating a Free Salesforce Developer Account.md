# Lesson 2 — Creating a Free Salesforce Developer Account

## Lesson Summary

This lesson explains how to create a **free Salesforce Developer Account** for learning and practice. Salesforce provides this account specifically for educational and development purposes. The account does **not require payment, credit card information, or a trial period**, allowing unlimited usage for learning. The lecture also demonstrates account verification, password setup, and login steps.

---

## Key Points

- Create a **new Developer Account** specifically for the course.
- Developer accounts are provided by **Salesforce for learning purposes**.
- No:
    - Payment required
    - Credit card required
    - Trial period limitation
- Signup is done through **developer.salesforce.com**
- Account activation requires **email verification**
- Username must follow **email format**
- Login is performed through **login.salesforce.com**

---

## Detailed Notes

### What is a Salesforce Developer Account?

A **Salesforce Developer Account** is a free environment provided by Salesforce to:

- Learn Salesforce features
- Practice configurations
- Build and test applications
- Perform hands-on training

### Important Characteristics

- **Free to use**
- **No credit card required**
- **No expiration period**
- Can be used indefinitely for learning

### Recommendation from the Instructor

Even if you already have an existing developer account:

- Create a **separate account for this course**
- Keep course activities isolated from personal experiments or previous work

## Steps / Process

### Step 1 — Open Salesforce Developer Website

Open browser and go to:
`developer.salesforce.com`

---

### Step 2 — Click Sign Up

On the homepage:
- Select **Sign Up**
- (Note: Login button is for existing users)

---

### Step 3 — Fill Registration Details

Required fields:
- First Name
- Last Name
- Email Address
- Job Role
- Company Name
- Country
- Postal Code
- Username

**Important:**
Your **email address** receives:
- Account verification link
- Password setup instructions

---

### Step 4 — Create Username

**Rules:**
- Username must follow **email format**

**Example:**
```
platformappbuilder@example.com
```

**Important:**
- Username does not necessarily need to match your email.
- Save the username somewhere safe.

---

### Step 5 — Accept Terms and Submit

- Check **Terms and Conditions**
- Click **Sign Me Up**

Salesforce processes the request.

---

### Step 6 — Verify Account Through Email

Open inbox and:
- Find **Salesforce verification email**
- Click **Verify Account**

**Note:**
- Wait a few minutes if email does not arrive immediately.

---

### Step 7 — Set Password

Password requirements mentioned:
- Minimum **8 characters**
- At least **1 letter**
- At least **1 number**

Create password and select **Change Password**.

---

### Step 8 — Access Salesforce Lightning UI

After password creation:
- Salesforce opens into **Lightning Experience UI**
- Initial navigation help may appear

You can access:
- Profile
- Settings
- Account information

---

### Step 9 — Login Later

To access account again, go to:
`login.salesforce.com`

Provide:
- Username
- Password
- Click **Log In**

**Note:**
- If the account was just created, wait **15–20 minutes** before first login if access fails.
